# 要件定義書

## 概要

3×3グリッドで思考を階層的に展開できる個人向けWebアプリ。
各セルをクリックするとさらに3×3に展開できる「無限階層式」のマンダラートツール。

---

## プラットフォーム

- **Web アプリ**（ブラウザで動作）
- レスポンシブ対応（PCメイン、スマホも考慮）
- 将来的に React Native（スマホ）・Tauri（デスクトップ）へ段階的に展開予定

---

## グリッド仕様

- **階層式（無限展開）**: どのセルもクリックして3×3に掘り下げ可能
- **並列展開**: 同じ階層に複数の3×3グリッドを作成できる
  - 画面には常に **1つのグリッドのみ** を表示する
  - 並列グリッドが存在する場合のみ **左右ナビゲーションボタン**（← →）が表示される
  - **「+ 新しいグリッドを追加」ボタン** で同じ階層に並列グリッドを作成
- 階層の深さ・並列数に制限なし
- セルにはテキスト・画像・背景色を設定できる

---

## 表示モード

| モード | 内容 |
|--------|------|
| **3×3 表示** | 現在の階層の1グリッドのみ表示。セル編集・階層移動のメイン画面 |
| **9×9 表示** | 現在のグリッド＋その直下サブグリッドを2階層まとめて表示。全体俯瞰に便利 |

**切り替え方法（両方対応）:**
- 画面上部のトグルボタン `[3×3] [9×9]` でいつでも切り替え可能
- ズームアウト操作でも自動的に 9×9 表示に切り替わる（ズームインで 3×3 に戻る）

## セル操作

### クリック・タップの割り当て

| 操作 | 動作 |
|------|------|
| シングルクリック / タップ | サブグリッドへ掘り下げ（階層移動） |
| ダブルクリック / ダブルタップ | セル編集画面を開く |
| 編集画面「確定」 | 変更を保存して閉じる |
| 編集画面「キャンセル」/ `Escape` | 変更を破棄して閉じる |

**エッジケース:**

| 状況 | 動作 |
|------|------|
| サブグリッドを持たない空セルをシングルクリック | 編集モードにフォールバック |
| 空の周辺セル（中心が空で無効化） | クリック操作すべて無効 |
| ルートグリッドの中心セル（空）をシングルクリック | 編集モード |
| ルートグリッドの中心セル（入力あり）をシングルクリック | ホーム（ダッシュボード）へ移動 |

### セル編集画面

**形式（レスポンシブ）:**

| 画面幅 | 形式 |
|--------|------|
| スマホ（〜768px） | ボトムシート（下からスライドアップ） |
| タブレット〜デスクトップ（769px〜） | モーダル（中央オーバーレイ） |

**内容:**
```
┌─────────────────────────┐
│  セルを編集             │
│  テキスト入力エリア     │
│  画像アップロード       │
│  背景色（プリセット）   │
│  [キャンセル]  [確定]   │
└─────────────────────────┘
```

**Tab キーによるセル間移動（デスクトップのみ）:**

```
セルのナンバリング:       Tab 移動順（ループ）:
┌───┬───┬───┐            5 → 8 → 7 → 4 → 1 → 2 → 3 → 6 → 9 → 5
│ 1 │ 2 │ 3 │
├───┼───┼───┤            中心（5）から開始し、外周を時計回りに一周
│ 4 │ 5 │ 6 │            Shift+Tab は逆順
├───┼───┼───┤
│ 7 │ 8 │ 9 │            中心セルが空の場合は Tab 押下でも移動しない
└───┴───┴───┘
```

## ドラッグ＆ドロップ

### 同一グリッド内 D&D

| ドラッグ元 | ドロップ先 | 挙動 |
|-----------|-----------|------|
| 周辺セル | 周辺セル | 階層ごと入れ替え（サブツリーを swap） |
| 中心セル | 入力ある周辺セル | 内容のみ入れ替え（text / color のみ） |
| 中心セル | 空の周辺セル | 階層全体をコピー（元は残る） |
| 入力ある周辺セル | 中心セル | 内容のみ入れ替え（対称ルール） |
| 空の周辺セル | 中心セル | 何もしない |

### 9×9 表示での D&D（デスクトップのみ）

サブグリッドをまたいだ D&D も可能。同じルールを適用する。

| 特記事項 |
|---------|
| ルートグリッドの中心セル（★）はドラッグ対象外 |
| ドラッグ中はドロップ可能なセルをハイライト表示 |

## 階層間・マンダラ間のデータ移動

### 1. カット＆ペースト

- 右クリック（長押し）コンテキストメニューから カット / コピー / ペースト
- キーボードショートカット: `⌘X` / `⌘C` / `⌘V`（`Ctrl` 版も対応）
- カット後は元セルがグレーアウト表示

### 2. ストックエリア

画面右端サイドパネルに常駐する一時保管エリア。

| 項目 | 仕様 |
|------|------|
| ドラッグ方向 | セル → ストックは常にコピー（元は残る） |
| ストック → セル | 同一グリッド内 D&D と同じルールを適用 |
| 永続性 | アカウントに紐づき永続保存（アプリを閉じても残る） |
| 上限 | なし（手動で削除） |

ストックの snapshot は grids + cells の JSON シリアライズ。元データが変わっても内容は変化しない。

---

## セル入力のバリデーション

| ルール | 内容 |
|--------|------|
| 中心セルが空 | 周辺セルは入力不可（非活性表示） |
| 周辺セルに入力あり | 中心セルをクリア不可（エラートーストを表示） |

**「空」の定義:** `text` が空文字 **かつ** `image_path` が NULL

---

## 空データの非保存ルール

| 状況 | 動作 |
|------|------|
| 新規マンダラートで未入力のまま離脱 | DB に保存しない |
| 既存グリッドの全セルが空になった | そのグリッドを自動削除 |
| ルートグリッドが空になった | マンダラート全体を自動削除 |
| 並列グリッドが空になった | その並列グリッドのみ削除 |

削除後は Undo トーストを表示（数秒以内に取り消し可能）。

---

## Undo / Redo

- `⌘Z` / `⌘Y`（Mac）・`Ctrl+Z` / `Ctrl+Y`（Windows）で汎用対応
- 対象: テキスト編集・色変更・D&D・削除・セル移動など

## マンダラート管理

### タイトル設定フロー

1. 新規作成時はタイトル未設定のままエディタへ入る
2. ホームへ離脱する際、タイトルが未設定であればダイアログを表示
3. ダイアログの初期値はルート中心セルのテキストを自動セット
4. 確認・編集して「保存」→ ダッシュボードへ
5. 後からダッシュボードでリネームも可能

タイトルはルート中心セルとは独立して管理（中心セルの変更はタイトルに影響しない）。

### ダッシュボード機能

- グリッドのミニプレビューを表示（カード形式）
- 検索: タイトル検索・セル内容の全文検索
- ソート: 更新日順
- 操作: 複製・削除・リネーム

## ナビゲーション

### パンくずリスト

- 常に画面上部に表示
- 各アイテム: セルのテキスト＋画像アイコン（画像のみのセルは🖼アイコン）
- 各アイテムにグリッドの **ミニプレビュー** を表示し、どのセルを掘り下げたかをハイライト表示

```
[ホーム]      >  [テーマA]
┌─┬─┬─┐        ┌─┬─┬─┐
│ │ │ │        │ │ │ │
├─┼─┼─┤        ├─┼─┼─┤
│ │ │■│        │■│ │ │   ■ = 現在のパス上のセル
├─┼─┼─┤        ├─┼─┼─┤
│ │ │ │        │ │ │ │
└─┴─┴─┘        └─┴─┴─┘
```

- 先頭「ホーム」をクリック → ダッシュボードへ（専用ボタンは設けない）

### 並列グリッドのナビゲーション

- 並列グリッドが存在するときのみ左右ボタン（← →）を表示
- ボタン非表示 = 並列グリッドなし

## グリッドの削除・設定

- **長押し（スマホ）/ 右クリック（デスクトップ）** でコンテキストメニューを表示
- メニュー項目: 削除・タイトル変更・複製

---

## メモ欄

- 各グリッド（3×3 ユニット）に紐付いたメモ
- **Markdown 記法**対応（見出し・箇条書き・太字など）
- 編集モード / プレビューモードを切り替え可能
- 自動保存（debounce）
- エディタ右端サイドパネルの「メモ」タブに配置（ストックエリアとタブ切り替え）

---

## データ保存・同期

- **Supabase（PostgreSQL）** によるクラウド保存
- **Supabase Realtime** による複数デバイス間リアルタイム同期
- **オフライン対応**: オフライン中もローカルに一時保存し、復帰時に自動同期
  - 実装: IndexedDB によるローカルキャッシュ
  - 画面上部にオフラインインジケーターを表示

---

## 認証

- メール + パスワード
- Google / GitHub OAuth
- Supabase Auth を使用

## エクスポート機能

- **対象**: 現在表示中のグリッド以下の全階層
- **PDF / PNG**: 現在の表示をそのまま出力（html2canvas + jsPDF）
- **JSON / CSV**: 構造化データとして出力

---

## インポート機能

### 対応形式

| 形式 | 説明 |
|------|------|
| アプリの JSON / CSV | エクスポートの完全往復対応 |
| インデントテキスト | スペース・タブのネストで階層を表現 |
| Markdown 見出し | `#` / `##` / `###` のレベルで階層を表現 |
| クリップボード | 上記いずれかを直接貼り付け |

**解析ルール:** 各階層のアイテムを周辺セル（最大8つ）に配置。9個以上は並列グリッドに自動分配。

### インポート先

- 新規マンダラートとして作成
- 既存セルに差し込む（そのセルの子グリッドとして挿入）

### UI フロー

```
[インポート] → ① 形式選択 → ② プレビュー確認 → ③ インポート先選択 → ④ 実行
```
